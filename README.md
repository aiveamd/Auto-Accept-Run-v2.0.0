# Auto-Accept-Run v2.0.0

自動接受 Antigravity Agent 的所有步驟 — 檔案修改、終端指令、自動補全。

## 安裝方法
1. 下載 `auto-accept-run-2.0.0.vsix`
2. 在 Antigravity 按 `Ctrl+Shift+P` → 輸入 `Extensions: Install from VSIX...`
3. 選擇下載的 `.vsix` 檔案
4. 重新載入視窗 (`Ctrl+Shift+P` → `Reload Window`)

## 使用方式
- 右下角狀態列會顯示 `✅ Auto-Accept: ON`
- 按 `Ctrl+Alt+Shift+U` 可切換開/關
- 點擊狀態列文字也可以切換
